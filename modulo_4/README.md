# Módulo 4 — Preparación para Linux embebido

Concurrencia, procesos y señales en C puro sobre Linux.
Base directa para programar sobre Raspberry Pi sin abstracciones.

## Ejercicios
- 1_fork: creación de procesos con fork() y sincronización con wait()
- 2_threads: concurrencia con pthreads y mutex
- 3_senales: manejo de señales del sistema con sigaction y cierre limpio de daemons
