# C Curso

Aprendizaje de C puro — desde fundamentos hasta Linux embebido en Raspberry Pi.

## Módulos
- Módulo 1: Fundamentos — compilación con gcc, Makefile, funciones separadas, memoria dinámica
- Módulo 2: Estructuras de datos — listas enlazadas, colas FIFO con doble puntero
- Módulo 3: Embedded — punteros a funciones, operaciones de bits, archivos, strings
- Módulo 4: Linux embebido — procesos, threads, señales, daemons
